ZdCommentsWidget allows you to display top commenters or last commenters in a widget

== Description ==

Display last commenters or top commenters
Display commenter name or not
Default gravatar
Gravatar size
Display link to commenters blog or not
Add no follow to the link or not

== Installation ==

1. Setup is very simple, just unzip the archive in your wp-content/plugins folder
2. You can now activate the plugin and add the widget to your sidebar.
